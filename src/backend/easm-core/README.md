# EASM Core

Shared libraries and utilities for the EASM backend.

This package contains reusable components, utilities, and core functionality that can be shared across different Django applications within the EASM platform.

## Structure

```
easm-core/
├── __init__.py
├── README.md
└── (future core utilities and shared libraries)
```

## Usage

Import shared utilities from this package in your Django applications:

```python
from easm_core import some_utility
```
